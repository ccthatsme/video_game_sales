# Video Game Sales Analysis

Data analysis project for exploring global video game sales trends.

## Project Structure

- `data/` - raw and processed datasets
- `notebooks/` - Jupyter notebooks
- `src/` - reusable Python code
- `outputs/` - charts, reports, exports

## Setup

1. Create and activate a virtual environment.
2. Install dependencies:

```bash
pip install -r requirements.txt
```

## Usage

- Run notebooks in `notebooks/` for exploratory analysis.
- Use scripts in `src/` for repeatable processing.

## Notes

- Large datasets and local environment files are ignored via `.gitignore`.
- Do not commit secrets (`.env`, keys, tokens).
